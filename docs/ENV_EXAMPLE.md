# Environment Example

```env
NEXT_PUBLIC_WEBSERBISYO_API_URL=https://webserbisyo.com
NEXT_PUBLIC_EVENT_SLUG=your-platform-event-slug
NEXT_PUBLIC_DESIGN_MODE=true
NEXT_PUBLIC_TEMPLATE_ID=wedding-custom-starter-v1
```

Do not add `custom_frontend_origin_url`, Supabase envs, database URLs, or secrets.

`NEXT_PUBLIC_*` values are browser-public and may be inlined at build time.
