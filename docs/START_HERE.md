# Start Here

This starter is for frontend-only custom wedding websites connected to the WebSerbisyo RSVP platform.

Read `ARCHITECTURE.md`, `DATA_CONTRACT.md`, `SECTIONS.md`, and `DO_NOT_BREAK.md` before making changes.

The default runtime should start from the platform-compatible renderer in `src/components/platform/*`. Do not replace it with a starter-only landing page, marketing navbar, hero layout, or client theme.

Use design mode for visual work. Use live mode only when testing a real published platform event slug.
