# Placeholders

Sample content is allowed only when `NEXT_PUBLIC_DESIGN_MODE=true`.

Live mode must never show sample names, venues, sponsors, gift details, or contact details.

Placeholder images are not required in the starter. Add real public-safe client assets only in a per-client repo.

Fake QR codes and fake photos are allowed only in design mode. Live mode must render real public media, text-only content, or a neutral missing-media state.
